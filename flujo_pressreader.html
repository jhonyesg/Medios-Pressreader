Flujo completo de Publimetro (use_browser_auth=True)

Publimetro tiene use_browser_auth = True en press.ini, lo que activa el refresco automático si todos los tokens estáticos fallan durante la descarga.

run_descarga.py Publimetro │ ├─ PASO 1: Valida tokens estáticos de press.ini │ ✅ Token válido → continúa al downloader directamente │ ❌ Token inválido → llama a BrowserAuthenticator.get_fresh_token() │ ✅ JWT fresco obtenido (420 chars) → guardado en press.ini │ ├─ PASO 2: pressreader_downloader.py Publimetro │ ├─ Valida token antes de descargar (también llama browser auth si falla) │ ├─ Descarga 40+ imágenes en imagenes/ │ ├─ OCR + PDFs individuales en pdf_paginas_ocr/ │ ├─ PDF final: Publimetro_20250520.pdf │ └─ FTP: /Publimetros/20250520/{imagenes/, pdf_paginas_ocr/, Publimetro_20250520.pdf}
